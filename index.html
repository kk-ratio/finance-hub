⚙️ Settings — Connect GitHub Storage
Your data is saved as a file (data.json) inside your own GitHub repo. You need a GitHub Personal Access Token so the page can read/write that file. This is a one-time setup.
  1. 1
    Go to github.com/settings/tokens → Generate new token (Fine-grained)
  2. 2
    Set Token name to anything (e.g. "Finance Hub"), Expiration to "No expiration"
  3. 3
    Under Repository access → select "Only select repositories" → pick finance-hub
  4. 4
    Under Permissions → Repository permissions → set Contents to "Read and write"
  5. 5
    Click Generate token, copy it, paste below.
🔒 Security: Your token is stored only in the URL hash fragment (#token=…) which never gets sent to any server.
